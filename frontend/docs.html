API Documentation

This page provides an overview of the available API endpoints for the Film Reviews Management system.

Movies API

1. Get All Movies

Method: GET

Endpoint: /movies

Description: Fetch all movies.

2. Get a Movie by ID

Method: GET

Endpoint: /movies/{id}

Description: Retrieve details of a specific movie by its ID.

3. Search for Movies

Method: GET

Endpoint: /movies/search?query={searchTerm}

Description: Search for movies by a search term.

4. Create a New Movie

Method: POST

Endpoint: /movies

Description: Add a new movie by providing its details.

5. Update a Movie

Method: PUT

Endpoint: /movies/{id}

Description: Update an existing movie by its ID.

6. Delete a Movie

Method: DELETE

Endpoint: /movies/{id}

Description: Delete a movie by its ID.

Reviews API

1. Get All Reviews

Method: GET

Endpoint: /reviews

Description: Retrieve all reviews.

2. Get Reviews for a Movie

Method: GET

Endpoint: /reviews/{movieId}

Description: Retrieve all reviews for a specific movie.

3. Add a New Review

Method: POST

Endpoint: /reviews/{movieId}

Description: Add a new review for a specific movie.

4. Update a Review

Method: PUT

Endpoint: /reviews/{id}

Description: Update an existing review by its ID.

5. Delete a Review

Method: DELETE

Endpoint: /reviews/{id}

Description: Delete a review by its ID.

Error Responses

400 - Bad Request

The request is invalid or missing required parameters.

404 - Not Found

The requested resource could not be found.

500 - Internal Server Error

An unexpected error occurred on the server.
